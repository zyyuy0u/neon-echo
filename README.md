# NEON ECHO

NEON ECHO is a browser-based third-person 3D exploration game set in a silent synthwave city.

## Commands

- `npm install` — install the pinned dependencies.
- `npm run dev` — start the Vite development server.
- `npm run build` — type-check and create a production build.
- `npm run preview` — serve the production build locally.
- `npm run lint` — run ESLint.
- `npm run typecheck` — run strict TypeScript checks without emitting files.
- `npm test` — run the Vitest unit suite once.
- `npm run e2e` — start Vite and run the Chromium Playwright smoke test.
