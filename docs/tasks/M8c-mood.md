# M8c 派工單：AAA 對標——氛圍與演出（P2，M8 最終批）

## GOAL（目標與背景）
依 `docs/AAA-BENCHMARK.md`（先讀）之 M8c 批次。基線：M8b 已簽收 commit 58c318a（全測試綠）。

實作內容：
1. **天色循環**：12 分鐘一輪的色相相位（合成波日落 ⇄ 深夜藍紫，**不出現白晝**，
   維持 palette 色域）；sky shader 漸層色、霧色、太陽亮度、環境光強度四者聯動；
   相位入存檔（round-trip）。**相位→顏色插值為純函式並單元測試**（0 相位=現行日落
   配色；0.5 相位=深夜配色；顏色恆在允許色域內——測試以 RGB 分量範圍斷言）。
   減少動態：循環照常（非動態眩暈源），但過渡速率不變、不加閃爍。
2. **碑文逐字演出**：開啟碑文→逐字顯示（每字 18ms；任意鍵立即補完；再按 E 關閉）+
   每 3 字一個合成 blip 音（音量隨主音量）；鏡頭輕推近 8%（0.4s ease，關閉還原；
   減少動態=無推近、文字直接全顯、blip 保留單次）。**逐字排程為純函式並單元測試**
   （給定文本與經過時間→應顯示的字元數；skip 行為）。
3. **選單動效**：面板開啟滑入+淡入 240ms、按鈕 focus/hover 輝光脈衝（CSS）、
   主選單背景加緩慢漂浮粒子（純 CSS 或低成本 canvas，不進 WebGL 預算）；
   減少動態=全部瞬時切換、粒子靜止。
4. **結局演出強化**：喚醒——全城燈光由近而遠波次點亮（4s，分 4 波，每波亮度上升+
   音層疊加）；安眠——由遠而近熄滅（4s，音樂淡出至單音）。以既有 particles/
   WorldBuilder 燈光元素分組實作，波次分組計算為純函式並測試（給定距離→波次索引）。
   減少動態=2 波、總長 1.5s。
